# Halt语句

用于强行终止应用程序的执行，返回操作系统（非正常退出方式）

基本和任务管理器结束进程差不多

# Exit 语句

　用于从当前代码块中退出。

　若该代码是主程序，则终止该程序。

　如果是函数或过程，则立即终止该函数或过程

# Abort 语句

直接终止调用所在的函数的代码块，退出整个调用过程

不像exit只退出自己的代码块

# Runerror语句

终止程序的执行，并产生运行错误（返回错误代码）



# Close()

正常退出程序

# Application.Terminate

直接终止程序，和Halt有异曲同工之效果
